# Privacy Policy – Hồi diễn cuối

**Effective date:** March 2026

Thank you for playing **Hồi diễn cuối**.

This game is designed to provide a simple puzzle experience inspired by classic point-and-click puzzle games. The developer respects user privacy and does not collect or process personal data.

---

## Data Collection

The game **does not collect, store, or transmit any personal information**.

Specifically, the game does not collect:

* Name
* Email address
* Location data
* Device identifiers
* Contacts
* Photos, media, or files
* Analytics or usage tracking data

The game also **does not connect to external servers** to transmit user data.

---

## Local Data Storage

The game **does not store user data locally** on the device.

No save files, preferences, or user information are stored by the application.

---

## Permissions

The game **does not request any Android system permissions**.

Because of this, the application does not access:

* Storage
* Camera
* Microphone
* Contacts
* Location
* Network data

---

## Third-Party Services

This game **does not use third-party services**, including:

* Advertising networks
* Analytics services
* Cloud storage services
* User account systems

---

## Children's Privacy

Since the game does not collect or process any user information, it is safe for players of all ages. No personal data from children or adults is collected.

---

## Changes to This Privacy Policy

If the privacy policy changes in the future, the updated version will be published with the game update.
